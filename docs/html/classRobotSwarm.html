Detailed Description

A ROS 2 Node that publishes a message with an incrementing count at regular intervals.

The RobotSwarm class initializes a ROS 2 node that publishes messages at a user-defined frequency. The messages contain an incrementing count to indicate the number of times the node has published. The frequency and topic name are configurable parameters.
