Detailed Description

This file contains unit tests for a simple ROS 2 node that tests the creation of a publisher.

This file uses Google Test (gtest) to test a basic ROS 2 node implementation. It focuses on the creation of a publisher and verifies that it has been successfully created by checking the number of publishers associated with a topic.
